# Library Management System (Python + MySQL)

This is a simple CLI based Library Management System built using Python and MySQL.

## Features
- Add student
- Delete student
- Search student
- Add books
- Issue book
- Return book
- Check available books

## Technologies Used
- Python
- MySQL
- PyMySQL
- Pandas

## Database
Tables used:

- student
- used to store information of students.
- ex : student_id,student_name and student_class
- 
- books
- used to stire information of books present in library.
- ex : book_id,book_name,author and number of books
- 
- issue_book
- stores the information of issued book and student that issue book.
- ex : date of issue,student_id,book_id and returning date.
